# 🎮 Learn with fun : Python Quiz Game 

A fully functional **Multiple Choice Quiz Game** built entirely using **Python**.
This project runs in the command line and tests users with 20 MCQ questions from various topics including general knowledge, science, mathematics, programming, and history.

It demonstrates the use of Python concepts such as:

* Lists and Dictionaries
* Loops and Conditional Statements
* Functions
* Input Validation
* Score Tracking

---

## 🚀 Features

* 📚 20 Predefined Multiple Choice Questions
* 🔁 Input validation (only accepts A/B/C/D)
* 🧮 Real-time score calculation
* 🏆 Performance-based final message
* 🧩 Clean and easy-to-understand code structure
* ➕ Easily expandable question bank

---

## 🛠️ Tech Stack

* **Language:** Python
* **Interface:** Command Line (Terminal / CMD)
* **Concepts Used:** Functions, Lists, Dictionaries, Loops, Conditionals

---

## ▶️ How to Run

1. Make sure Python is installed on your system.
2. Save the code in a file (for example: `quiz_game.py`).
3. Open terminal or command prompt in that folder.
4. Run:

```bash
python quiz_game.py
```

---

## 📌 How the Game Works

* The function `run_quiz()` stores all questions inside a list of dictionaries.
* Each question contains:

  * The question text
  * Four options (A, B, C, D)
  * The correct answer
* The user must enter a valid option (A/B/C/D).
* The program checks the answer and updates the score.
* At the end, the final score is displayed along with a performance message.
